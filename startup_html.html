HTML Attributes

  • All HTML elements can have attributes.
  • Attributes provide additional information about elements.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs, like name = "value".

Links are defined with 'a', where the link is included in the start tag after 'a', and the element text includes the hyperlinked text.

HTML images

Images are defined with the 'img' tag and have a number of attributes, such as 'src' (the source file), 'alt' (alternative text for the image), 'width', and 'height'.

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.
